# Extra Animation Options
To access these features, simply check the "Extra Options" button in the sprite viewer menu. Each of these options on affect the selected sprite, not all sprites at once. This page does not cover the "Event Editor" as that is covered [here](https://github.com/GrouganDev/PNGTuber-Plus-Studio/blob/master/docs/AnimationEvents.md)
## Randomize Animation
Randomizes the order in which animation frames are played for animated sprites that have this enabled.
![ezgif com-resize (1)](https://github.com/user-attachments/assets/36a541c1-278f-4894-b703-41e65b4afbad)


## Randomize Speed
Changes the animation speed of a sprite to a new random value every time a new frame in the animation is displayed. 
![ezgif com-resize](https://github.com/user-attachments/assets/16a5d15d-4130-4ae3-9fd2-6a29ab84849f)

The minimum and maximum random values can be configured by the user through the two sliders underneath "Randomize speed" checkbox.

## Reset Animation on Costume Change
Every time the user switches costumes, the animated sprites goes back to their first indexed frame, and their internal animation tick is also brought back to zero.
